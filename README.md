# 🏠 Delhi House Price Prediction

## 📌 Overview

This project focuses on predicting house prices in Delhi using Machine Learning techniques. The goal is to build a model that can estimate property prices based on features such as location, area, number of bedrooms, and other relevant factors.

---

## 🚀 Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature selection and engineering
* Model training using regression algorithms
* Performance evaluation using metrics
* Price prediction for new inputs

---

## 📂 Dataset

The dataset contains housing information such as:

* 📍 Location
* 📏 Area (in sq. ft.)
* 🛏 Number of bedrooms
* 🛁 Number of bathrooms
* 🏢 Property type
* 💰 Price

> Dataset can be collected from sources like Kaggle or real estate websites.

---

## 🛠️ Technologies Used

* Python 🐍
* NumPy
* Pandas
* Matplotlib / Seaborn
* Scikit-learn

---

## ⚙️ Machine Learning Models

* Linear Regression
* Decision Tree Regressor
* Random Forest Regressor

---

## 📊 Model Evaluation

We evaluate the model using:

* Mean Absolute Error (MAE)
* Mean Squared Error (MSE)
* R² Score

---

## 🔍 Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Model Evaluation
7. Prediction

---

## 📈 Results

The model successfully predicts house prices with reasonable accuracy. Random Forest performed better compared to other models in most cases.
